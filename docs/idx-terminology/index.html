Terminology

DID#

Decentralized Identifiers (DIDs) are globally unique persistent identifiers as defined by the W3C DID specifications.

3ID#

A 3ID is a DID document using the 3 method name and stored by the Ceramic network.

A 3ID notably stores the DocID of the Root Index used by the IDX library.

Ceramic#

In this documentation, Ceramic can refer to either the Ceramic network itself, or an instance of a Ceramic API used by the the IDX library, as exposed by the js-ceramic repository.

Document#

A Document is a representation of structured data on the Ceramic network along with metadata. The IDX library provides high-level interfaces interacting with these documents.

DocID#

A Document Identifier (DocID) is a URL representing a Document. Multiple APIs in the IDX library use DocIDs as inputs and/or outputs.

Identity Index / IDX#

The Identity Index and IDX can sometimes both be used to refer to the Identity Index specification CIP, but in this documentation the term Identity Index will always be used to refer to the specification CIP, while IDX refers to the library, or possibly the IDX class in code snippets.

Schema#

A JSON schema may be used to validate the structure of a Document. These schemas may be defined in Ceramic CIPs, other specifications or created by developers for their application needs.

Definition#

A Definition is a specific type of Document used by the IDX library to represent other documents in the Root Index.

A Definition is based on a specific Schema describing the document it represents, notably the Schema used to validate the document.

Entry#

An Entry is a simple data structure containing of a ref DocID for the content Document, and a set of string tags used for content discovery.

Root Index#

The Root Index is the main Document used by the IDX library to access and store the contents of the Identity Index.

The Root Index is a mapping of a Definition, represented by its DocID, to an Entry.

The IDX library provides various APIs to interact with the Root Index and the associated Definitions and Entries.
